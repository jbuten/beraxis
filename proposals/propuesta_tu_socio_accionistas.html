Origen
Qué es Tu Socio
Tu Socio es una plataforma de crédito digital diseñada para operar en el ecosistema de distribución mayorista. Permite a comercios y compradores acceder a financiamiento ágil, gestionado completamente en línea, con trazabilidad financiera total y experiencia moderna tanto para el cliente como para el equipo operativo.

La plataforma nace como evolución de un producto con cartera activa demostrada. Los clientes existentes serán migrados de forma ordenada al nuevo sistema, firmando un nuevo contrato bajo la nueva empresa, sin interrumpir operaciones en ningún momento del proceso.

Infraestructura del Sistema
Componentes Tecnológicos
Infraestructura requerida para operar Tu Socio
La operación segura, estable y escalable de Tu Socio requiere una infraestructura distribuida en múltiples componentes. Cada servidor, servicio e integración cumple una función específica e irremplazable dentro del ecosistema.
Google Cloud Platform — Servidores y Cómputo
Base de Datos — Producción · Cloud SQL for PostgreSQL
Instancia administrada Cloud SQL (db-custom-4-8192) — almacena toda la información financiera: clientes, créditos, cuotas, pagos y auditoría con integridad transaccional.
Base de Datos — Réplica y Backup · Cloud SQL Read Replica
Réplica administrada en tiempo real para alta disponibilidad. Backups automáticos diarios cifrados con retención de 30 días gestionados por GCP.
Backend — API Principal · Compute Engine e2-standard-4
VM de cómputo optimizado en GCP. Motor principal del sistema: créditos, pagos, autenticación, reglas financieras e integraciones externas.
Frontend — Backoffice Web · Compute Engine e2-standard-2
VM en GCP sirviendo el panel de control operativo en React: gestión de clientes, originación, cobros, conciliación y reportes.
Workers — Procesamiento Asíncrono · Compute Engine e2-standard-2
VM dedicada a tareas en segundo plano: cálculo de mora, cierre de cuotas, reportes automáticos, recordatorios y reconciliaciones periódicas.
Automatización n8n · Compute Engine e2-standard-2
VM en GCP para orquestación de flujos: notificaciones, alertas operativas, disparadores de eventos e integraciones entre servicios.
Comunicación — WhatsApp Business API · Compute Engine e2-medium
VM en GCP para gestión de mensajería con clientes: recordatorios de pago, confirmaciones de cuota, alertas de mora y soporte automatizado.
Almacenamiento de Archivos · Google Cloud Storage
Bucket Cloud Storage Standard (500 GB) para documentos KYC, contratos firmados, comprobantes de pago e identidades verificadas, con cifrado en reposo.
Monitoreo y Observabilidad · Google Cloud Monitoring
Cloud Monitoring + Cloud Logging para supervisión en tiempo real de todos los servicios, alertas automáticas ante incidentes y métricas de rendimiento.
Proxy y Balanceador · Cloud Load Balancing + Nginx
Google Cloud Load Balancing con Nginx en Compute Engine: enrutamiento de tráfico, terminación SSL, seguridad perimetral y exposición controlada de servicios.

Servicios e Integraciones
Conectividad
Servicios e Integraciones Externas
La plataforma se conecta a servicios especializados para asegurar la verificación de usuarios, la evaluación de buró de crédito en tiempo real y el mantenimiento de canales de comunicación institucionales.
Integraciones Externas
Equifax Dominicana — Buró de Crédito
Consulta de historial crediticio en cada solicitud. El resultado se almacena como snapshot para auditoría, historial y cumplimiento regulatorio.
MetaMap — Verificación de Identidad (KYC)
Verificación digital del cliente durante el proceso de alta: cédula, biometría facial y validación documental en tiempo real.
Pasarela de Pagos
Procesamiento de pagos digitales: tarjetas, transferencias y billeteras electrónicas, con conciliación automática contra el sistema de cuotas.
Correo Transaccional
Envío automatizado de notificaciones: bienvenida, confirmación de crédito, estados de cuenta, recordatorios de pago y alertas de vencimiento.
Envío de SMS — Notificaciones y Alertas
Mensajes de texto para recordatorios de cuota, alertas de mora, códigos de verificación y notificaciones críticas a clientes sin acceso a WhatsApp o correo.
Identidad Digital y Comunicaciones
Dominio Principal y Subdominios
Dominio corporativo + subdominios dedicados: api · app · admin · docs — cada servicio con su propio punto de acceso controlado.
Certificados SSL / TLS
Cifrado HTTPS en todos los endpoints y subdominios. Renovación automática gestionada sin intervención manual.
CDN y Protección — Cloudflare
Protección DDoS, enrutamiento global, caché de contenido estático y firewall de aplicaciones web (WAF) para todos los servicios expuestos.
Google Workspace — Correos y Colaboración
Correos institucionales del equipo bajo el dominio corporativo. Suite de colaboración: Drive, Meet, Docs y Calendar para la operación interna.

Seguridad y Gobernanza
Gobernanza
Seguridad y Respaldo Técnico
Para garantizar la seguridad de la información financiera y la continuidad operativa, el sistema implementa políticas estrictas de control de versiones, despliegue automatizado y respaldo diario.
Gestión, Seguridad y Gobernanza
Repositorio de Código — GitHub Privado
Control de versiones, historial de cambios y protección de la propiedad intelectual del sistema. Acceso controlado por roles.
CI/CD — Despliegue Continuo
Pipeline automatizado de validación y despliegue. Cada cambio aprobado se publica a producción con verificación previa en staging.
Gestión de Secretos y Ambientes
Manejo seguro de credenciales y API keys por ambiente separado: desarrollo, staging y producción. Sin credenciales expuestas en código.
Auditoría y Trazabilidad
Registro inmutable de cada acción crítica: quién, cuándo y qué cambió. Esencial para cumplimiento financiero y resolución de disputas.
Backups Automáticos y Plan de Recuperación
Respaldos diarios cifrados con retención de 30 días. Procedimiento documentado y probado de restauración ante fallo crítico.
